# OpenWeather
This is a simple weather application which is based on Jetpack Compose and implements a MVI pattern with modularisation.

Used technologies:
- Jetpack Compose
- Room DB
- Retrofit2
- Coroutines + Flow
- Koin DI
- Accompanist
- Animated Content

Will be implemented:
- User location
- Share (image)
