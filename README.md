# OpenWeather
This is a simple weather application which is based on Jetpack Compose and implements a MVI pattern with modularisation.
Implemented russian and english languages.

Used technologies:
- Jetpack Compose
- Room
- Retrofit2
- Coroutines + Flow
- Koin
- Accompanist
- Animated Content

Will be implemented:
- User location
- Share (image)
